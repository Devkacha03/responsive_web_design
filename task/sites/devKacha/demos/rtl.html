Overview

By adding rtl:true Owl will change direction from Right to left.

Default:

           rtl: false

Setup

                    $('.owl-carousel').owlCarousel({
                        rtl:true,
                        loop:true,
                        margin:10,
                        nav:true,
                        responsive:{
                            0:{
                                items:1
                            },
                            600:{
                                items:3
                            },
                            1000:{
                                items:5
                            }
                        }
                    })
