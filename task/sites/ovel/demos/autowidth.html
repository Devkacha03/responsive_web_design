Overview

Use width style on elements to get the result you want. If using with infinity loop add option 'items' more than 1. It all depends on the width of your content.

Setup

                    $('.owl-carousel').owlCarousel({
                        margin:10,
                        loop:true,
                        autoWidth:true,
                        items:4
                    })

html
